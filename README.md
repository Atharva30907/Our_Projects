# 🌱 Project Management System

A modern web-based application built using **Python (Flask)** that allows NGOs to manage and showcase their projects efficiently.

---

## 🚀 Features

- 🔐 Admin Login System
- ➕ Add New Projects
- 🗑️ Delete Projects
- 🖼️ Image Upload Support
- 📍 Project Location Tracking
- 📊 Dynamic Project Display
- 📄 Detailed Project View Page
- 🎨 Clean and Responsive UI

---

## 🛠️ Technologies Used

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Database:** SQLite
- **Templating Engine:** Jinja2

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/ngo-project.git
```
2. Navigate to project folder:
```
cd ngo-project
```
3. Install dependencies:
 ```
pip install flask flask_sqlalchemy
```
4. Run the application:
```
python app.py
```
5. Open browser:
```
http://127.0.0.1:5000/
```
